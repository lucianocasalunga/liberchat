## 🎨 Padrão de Uso

Todos os ícones seguem o padrão Lucide Icons (similar SF Symbols):
- Formato: SVG
- Tamanho: 24x24 viewBox
- Stroke: currentColor (herda cor do texto)
- Fill: none (outline style)

## 🔗 Uso nos Projetos

### LiberMedia
```html
<img src="/icons/image.svg" class="w-5 h-5" alt="Imagem">
```
